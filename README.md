# Juliana Lopes - The Vegan Chef Website

This is the official website for Juliana Lopes, a passionate plant-based chef specializing in transformational food experiences for wellness retreats and events.

## About The Project

This website serves as a digital portfolio for Juliana, showcasing her culinary philosophy, services, and delicious creations. It's designed to be a warm, inviting, and easy-to-navigate space for potential clients to learn more about her work and get in touch.

The site features:
- A beautiful homepage introducing Juliana and her offerings.
- An "About Me" page detailing her journey and brand values.
- Dedicated pages for "Retreat Catering" and "Treats & Bakes" (Pep Me Up).
- A comprehensive contact page.

## Built With

*   [Vite](https://vitejs.dev/)
*   [React](https://reactjs.org/)
*   [TypeScript](https://www.typescriptlang.org/)
*   [Tailwind CSS](https://tailwindcss.com/)
*   [Framer Motion](https://www.framer.com/motion/)

## Getting Started

To get a local copy up and running, follow these simple steps.

1.  Install NPM packages
    ```sh
    npm install
    ```
2.  Run the development server
    ```sh
    npm run dev
    ```

The application will be available at `http://localhost:5173` (or another port if 5173 is in use).
